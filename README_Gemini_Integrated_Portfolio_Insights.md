# Gemini Integrated Portfolio Insights

### Purpose
This notebook demonstrates a **full-stack AI-assisted portfolio analysis pipeline**, designed to provide **investors and analysts with actionable insights**, while highlighting advanced **data engineering, visualization, and AI reasoning skills**.

---

### Workflow Overview
1. **Load Portfolio Data**  
   - Ingest current positions with price, market value, allocation %, and gain/loss %.  
   - Clean and structure the data for analysis.

2. **Portfolio Visualization**  
   - Interactive allocation donut chart.  
   - Gain/Loss scatter and bubble charts.  
   - Multi-step rebalancing scatter chart.

3. **AI-Assisted Recommendations**  
   - Detect high concentration risk.  
   - Identify top performers and underperformers.  
   - Generate confidence-aware actionable recommendations.

4. **Multi-Step Rule-Grounded Rebalancing**  
   - Combine AI insights with financial rules:  
     - Reduce positions >20% allocation  
     - Monitor underperformers  
     - Consider taking profit on extreme gains  
     - Limit exposure for positions >25% of portfolio

5. **Professional Reporting**  
   - Inline interactive charts in Colab  
   - Conditional formatting in summary table  
   - Downloadable HTML report for presentation

6. **Conclusion & Key Takeaways**  
   - High concentration risks flagged  
   - Performance extremes highlighted  
   - Multi-step reasoning applied  
   - Portfolio management decisions clearly summarized

- **Data Engineering & Cleaning**: Transforming raw portfolio data for analysis  
- **Interactive Visualization**: Plotly charts with dynamic sizing and color coding  
- **Applied AI & Multi-Step Reasoning**: Confidence-aware recommendations  
- **Financial Logic Integration**: Rule-based portfolio rebalancing  
- **Reporting & Automation**: Conditional formatting and downloadable HTML report
