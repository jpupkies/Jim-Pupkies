# Gemini Prompt Demo Notebook

This notebook demonstrates automated batch prompt testing across multiple AI models using Google Colab. It is portfolio-ready and includes example outputs via screenshots.

## Overview

The notebook showcases:

- Secure API key handling
- Batch prompt execution workflow
- Multi-model support structure
- Screenshot-based demonstration of interactive workflow
- Clean documentation for portfolio presentation

> **Note:** Interactive widgets were removed for GitHub compatibility. All example outputs are preserved as screenshots.

## Running the Notebook

To run the notebook interactively:

1. Open `Gemini_Prompt_Demo.ipynb` in [Google Colab](https://colab.research.google.com/).  
2. Mount your Google Drive if needed for permanent screenshot syncing.  
3. Run the notebook cells in order.  

> The interactive interface is available in Colab, but GitHub shows pre-generated outputs via screenshots for portfolio purposes.
