# Portfolio Analysis Notebook

**Overview:** Multi-step portfolio analysis with stock positions, simulated news sentiment, risk scoring, and recommendations. Includes professional visualizations and inline screenshots.

**Features:**  
- Load positions (name, symbol, shares, % portfolio)  
- Simulate sentiment (positive/neutral/negative)  
- Buy/Hold/Sell recommendations  
- Risk assessment (High/Medium/Low)  
- Top-risk positions highlighted  
- Visuals: pie chart (% portfolio), horizontal bar chart (risk)  
- Inline screenshots per step

**Pipeline:** 1) Load positions 2) Simulate sentiment 3) Generate recommendations 4) Assess risk 5) Portfolio summary 6) Visualizations 7) Final evaluation
